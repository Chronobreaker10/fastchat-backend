# FastChat API

Чат-приложение с реальной архитектурой на **FastAPI**, **SQLAlchemy** и **PostgreSQL**. Реализована полная аутентификация (JWT), авторизация, управление чатами и обмен сообщениями.

## 🚀 Возможности (Use Cases)

- **Регистрация и вход пользователей** — создание аккаунта, вход по username, выдача JWT-токенов
- **JWT-аутентификация** — защита эндпоинтов с помощью `access` токенов
- **Управление чатами**:
  - Создание нового чата
  - Просмотр списка чатов текущего пользователя
  - Просмотр сообщений конкретного чата
  - Добавление / удаление участников чата
  - Удаление чата (с проверкой прав владельца)
- **Отправка сообщений**:
  - Отправка текстовых сообщений в чат
  - Просмотр истории сообщений (с пагинацией)
  - Удаление собственного сообщения
  - Редактирование сообщения
- **Авторизация** — только участники чата могут читать/писать сообщения, только создатель может удалять чат
- **Опционально:** уведомления о прочтении

## 🛠️ Используемые технологии

| Технология            | Назначение                                      |
|-----------------------|-------------------------------------------------|
| **Python 3.14**       | язык разработки                                 |

| **pwdlib / argon2**   | хеширование паролей                             |
| **uvicorn**           | ASGI-сервер                                     |
| **pydantic-settings** | управление переменными окружения                |
| **pytest**            | тестирование приложения                         |
| **ruff**              | форматирование и линтинг кода                   |
| **uv**                | управления зависимостями                        |
| **docker**            | запуск и развертывание                          |

## ✨ Запуск проекта

Генерация закрытого ключа: 
```bash
openssl genrsa -out private.pem 2048
mv ./private.pem ./secret-keys/
```

Генерация открытого ключа из приватного:
 ``` bash  
openssl rsa -in private.pem -pubout -out public.pem
mv ./public.pem ./secret-keys/
```

Запуск Docker контейнеров
```bash
just docker-up
 ```
