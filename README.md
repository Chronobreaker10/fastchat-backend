# FastChat API

Чат-приложение с распределенной микросервисной архитектурой на **FastAPI**, **SQLAlchemy** и **PostgreSQL**. Реализована полная аутентификация и авторизация через **JWT** токены, используется асимметричный алгоритм **RS256** для проверки валидности токена с помощью публичного ключа на любом микросервисе, управление чатами и обмен сообщениями в режиме реального времени с помощью вебсокетов. Для обмена сообщениями между микросервисами (чат и сервис уведомлений) используются **Faststream** и **Kafka** в роли брокера сообщений. Для хранений сессий пользователя, хэша Refresh токенов, информации об активных вебсокет-подключений к чатам, ограничения количества таких одновременных подключений на пользователя и связи между несколькими воркерами приложения (pubsub рассылка сообщений чата между экземплярами) используется **Redis**.

## 🚀 Возможности (Use Cases)

- **Авторизация и аутентификация**:
  - Регистрация нового пользователя
  - Вход в систему по логину и паролю (выдача Access и Refresh токенов)
  - Получение информации о текущем пользователе
  - Обновление токенов
  - Выход из системы
- **Управление чатами**:
  - Создание нового чата
  - Просмотр списка чатов текущего пользователя с последним сообщением
  - Просмотр сообщений и участников конкретного чата
  - Добавление новых участников в чат по username
  - Создание ссылки-приглашения в чат
  - Удаление чата (с проверкой прав владельца)
  - Исключение участников из чата (с проверкой прав владельца)
- **Отправка сообщений**:
  - Отправка текстовых сообщений в чат в режиме реального времени
  - Просмотр истории сообщений (с пагинацией - бесконечная прокрутка)
  - Удаление собственных сообщений
  - Редактирование собственных сообщений
  - Отметки об отправке, доставке и прочтении сообщений
- **Уведомления**:
  - Реализованы в отдельном микросервисе
  - Авторизация по JWT токену (проверка с помощью публичного ключа RS256)
  - Общение с основным приложением чата через faststream kafka

## ✨ Запуск проекта

Генерация пары ключей: 
```bash
openssl genrsa -out private.pem 2048
openssl rsa -in private.pem -pubout -out public.pem

```

Создание папки для хранения ключей:
 ``` bash  
mkdir secret_keys
mv ./private.pem ./secret_keys/
mv ./public.pem ./secret_keys/
```

Создание docker-сети:
```bash
docker network create fastchat-net
 ```

Запуск Docker контейнеров
```bash
just docker-up
 ```
